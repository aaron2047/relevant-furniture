# Relevant Furniture — Philadelphia

> A high-end, single-page website built for [Relevant Furniture](https://www.instagram.com/relevantfurniture1/) at 5675 N Front St, Philadelphia PA 19120.

**Live site:** [aaron2047.github.io/relevant-furniture](https://aaron2047.github.io/relevant-furniture)

---

## About the project

Relevant Furniture is one of Philadelphia's premier furniture destinations — 47K+ Instagram followers, same-day delivery, and a reputation built on exceptional service. This site was designed to match that energy: editorial, luxury-forward, and built for the North Philly community they serve.

Designed and developed by **[Woolph Marketing](https://woolph.com)** — a Philadelphia-based digital marketing agency.

---

## What's included

- Full single-page website (`index.html`) — no frameworks, no build tools, no dependencies
- Smooth-scroll navigation with sticky header
- Animated hero with illustrated furniture SVG
- Product category grid (Living Room, Bedroom, Dining, Accents)
- Featured product showcase with illustrated product cards
- Customer testimonials pulled from real Google reviews
- **Live financing calculator** — quick-pick presets, adjustable down payment, 5 rate tiers, 4 loan terms, real-time payment breakdown with principal/interest bar
- Store hours, address, and phone number (click-to-call)
- Full footer with social links
- Fully responsive down to mobile
- Google Fonts (Cormorant Garamond + Inter) — loads from CDN, no local assets needed

---

## Stack

| Layer | Choice |
|---|---|
| Markup | Semantic HTML5 |
| Styling | Vanilla CSS with custom properties |
| Interactivity | Vanilla JavaScript (no libraries) |
| Fonts | Google Fonts CDN |
| Hosting | GitHub Pages |

Zero dependencies. Opens in any browser. File size: ~72KB.

---

## Local preview

No build step needed — just open the file:

```bash
open index.html
```

Or serve it locally:

```bash
npx serve .
```

---

## Deploying updates

1. Edit `index.html`
2. Commit and push to `main`
3. GitHub Pages redeploys automatically within ~60 seconds
